Intro Fighters

Game project

Made using Unity, C#

Intro Fighters is a Twitch.tv integrated overlay game made with Unity.

In the game, viewers can take part in a fighting minigame that is shown on the broadcast as an overlay. Joining is done by typing in the stream's chat room. When a viewer joins, their game character will appear on stream for everyone to see. The characters are controlled automatically, so the viewers only need to watch and see the outcome.

Intro Fighters is most commonly used during quiet times while streaming, such as intro screens, breaks, etc. Basically it's meant to entertain when nothing else is happening.

The Twitch IRC client component of the game is also written by me from scratch. It is open-source, available on my GitHub.

The game is designed to work alongside the popular streaming software, OBS.

Because the game's main focus is stream integration, the target audience is mainly streamers. Although you can still try it out even if you are not a streamer yourself.

BotConversation

IOT Project

Made using Arduino, Unity, C#

BotConversation was a livestreaming project about two AI bots talking to each other. It was livestreamed for a few days on Twitch.tv late 2018.

The conversation intelligence was powered by Cleverbot API and the voices were generated using Google Cloud Text-to-Speech API

I used two 8x8 LED matrices for the bot face animations. They were both controlled by a single Arduino board. All the conversation logic, API calls and additional UI was made with Unity and C#

The project also included an automatic voting system where viewers could vote on trending topics for the bots to talk about. Voting was done via the stream chat.

The project is no longer active but you can still view past highlights that viewers of the stream have saved.
