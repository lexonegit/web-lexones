Intro Fighters

Game project

Made using Unity

Intro Fighters is a Twitch.tv integrated overlay game. It uses the Twitch IRC interface to allow viewers join as "on-screen" game characters via chat messages on whatever channel the game is being utilized on.

The game is designed to work alongside the popular OBS streaming software and it's mainly targeted towards streamers, although you can still try it out even if you are not a streamer yourself.

BotConversation

IOT Project

Made using Arduino and Unity

BotConversation was a project about two AI bots having an endless conversation. It was livestreamed for a few days on Twitch.tv late 2018.

The conversation intelligence was powered by Cleverbot API and text to speech was done with Google Cloud Text-to-Speech API

The physical robot faces were controlled by an Arduino and the interface shown on stream was made with Unity

Viewers could also influence the conversation by voting on trending topics for the bots to talk about every 10 minutes or so.

The project is no longer active but you can still view past highlights that viewers of the stream have made.
